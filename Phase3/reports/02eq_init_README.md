## 3. 탐색적 데이터 분석 (EDA)

### 3.1 기술통계량

Twitch 채널 데이터의 주요 통계량:

| 변수 | 평균 | 중앙값 | 표준편차 | 왜도 |
|------|------|--------|----------|------|
| 시청시간(시간) | ~2.5M | ~1.2M | ~3.8M | 양의 왜도 (우측 꼬리) |
| 팔로워 | ~1.8M | ~1.1M | ~2.5M | 양의 왜도 |
| 평균시청자 | ~8K | ~5K | ~12K | 양의 왜도 |

**해석**: 모든 주요 변수가 강한 양의 왜도를 보여 소수의 상위 스트리머가 대부분의 시청시간을 차지하는 **파레토 분포** 특성을 나타냄.

### 3.2 정규성 검정

| 변수 | D'Agostino p-value | 결론 |
|------|-------------------|------|
| 시청시간 (원본) | < 0.001 | 정규성 X |
| 시청시간 (로그변환) | > 0.05 | 정규성 근사 |
| 팔로워 (원본) | < 0.001 | 정규성 X |

**해석**: 원본 데이터는 정규분포를 따르지 않으나, 로그 변환 시 정규분포에 근접. 이는 데이터가 **로그정규분포(Log-normal distribution)**를 따름을 의미.

## 4. 상관관계 분석

### 4.1 피어슨 상관계수 (선형 관계)

주요 상관관계:

| 변수 쌍 | 상관계수 (r) | p-value | 해석 |
|---------|-------------|---------|------|
| 시청시간 ↔ 팔로워 | 0.85+ | < 0.001 | 강한 양의 상관 |
| 평균시청자 ↔ 최고시청자 | 0.90+ | < 0.001 | 매우 강한 양의 상관 |
| 스트리밍시간 ↔ 팔로워획득 | 0.45 | < 0.001 | 중간 양의 상관 |

### 4.2 스피어만 상관계수 (비선형 관계)

스피어만 상관계수는 순위 기반으로 계산되어 **비선형 단조 관계**도 포착 가능.

**해석**:
- 피어슨과 스피어만 상관계수가 유사 → 선형 관계 확인
- 스피어만이 더 높은 경우 → 비선형 단조 관계 존재
- 두 계수 모두 높음 → 변수 간 강력한 연관성

### 4.3 히트맵 해석 방법

```
상관계수 해석 기준:
  |r| > 0.7  : 강한 상관
  0.4 < |r| ≤ 0.7 : 중간 상관
  0.2 < |r| ≤ 0.4 : 약한 상관
  |r| ≤ 0.2 : 거의 없음
```

## 5. 회귀분석 및 예측

### 5.1 단순 선형 회귀 (시청시간 추세)

**모델**: `시청시간 = α + β × 연도`

| 지표 | 값 | 해석 |
|------|-----|------|
| 기울기 (β) | ~2.5B/년 | 연간 25억 시청시간 증가 |
| R² Score | 0.95+ | 모델이 95% 이상 설명력 보유 |
| RMSE | ~1.5B | 평균 15억 시청시간 오차 |

**미래 예측**:
- 2025년: ~22B 시청시간
- 2027년: ~27B 시청시간

### 5.2 다중 회귀 분석 (OLS)

**모델**: `Log(시청시간) = β₀ + β₁×Log(스트리밍시간) + β₂×Log(팔로워) + β₃×Log(평균시청자)`

| 변수 | 계수 | p-value | 해석 |
|------|------|---------|------|
| 스트리밍시간 | 0.3~0.4 | < 0.001 | 스트리밍 1% 증가 → 시청시간 0.3~0.4% 증가 |
| 팔로워 | 0.2~0.3 | < 0.001 | 팔로워 1% 증가 → 시청시간 0.2~0.3% 증가 |
| 평균시청자 | 0.5~0.6 | < 0.001 | 가장 큰 영향력 |

**해석**: 평균 시청자 수가 시청시간에 가장 큰 영향을 미치며, 이는 **콘텐츠 품질과 시청자 유지**가 성공의 핵심임을 시사.

## 6. 가설검정

### 6.1 독립표본 t-검정: 파트너 vs 비파트너

**가설**:
- H₀: 파트너와 비파트너의 시청시간에 차이가 없다
- H₁: 파트너와 비파트너의 시청시간에 차이가 있다

| 검정 | 통계량 | p-value | 결론 |
|------|--------|---------|------|
| Levene 등분산 검정 | - | < 0.05 | 등분산 가정 불충족 |
| Welch's t-test | 높음 | < 0.001 | H₀ 기각 |
| Mann-Whitney U | 높음 | < 0.001 | 비모수 검정도 동일 결론 |
| Cohen's d | > 0.8 | - | 큰 효과 크기 |

**결론**: 파트너 스트리머의 시청시간이 **통계적으로 유의하게 높음** (효과 크기 큼)

### 6.2 일원배치 ANOVA: 언어별 시청시간

**가설**:
- H₀: 모든 언어 그룹의 평균 시청시간이 동일하다
- H₁: 적어도 하나의 그룹이 다르다

| 검정 | 통계량 | p-value | 결론 |
|------|--------|---------|------|
| ANOVA F-test | 높음 | < 0.001 | 그룹 간 유의한 차이 |
| Kruskal-Wallis | 높음 | < 0.001 | 비모수 검정도 동일 |
| Tukey HSD | - | 다양 | 사후검정으로 구체적 차이 확인 |

**해석**: 영어 스트리머가 가장 높은 시청시간을 기록하며, 한국어, 스페인어 순으로 나타남.

### 6.3 카이제곱 검정: e스포츠 인기 vs 트렌드 상태

| 지표 | 값 | 해석 |
|------|-----|------|
| χ² 통계량 | 높음 | 기대빈도와 관측빈도 차이 큼 |
| p-value | < 0.05 | 두 변수는 독립적이지 않음 |
| Cramér's V | 0.3~0.5 | 중간 수준의 연관성 |

**결론**: e스포츠 인기도와 게임의 트렌드 상태는 **통계적으로 유의한 연관성**이 있음.

## 7. 시계열 분석

### 7.1 시계열 분해 (Seasonal Decomposition)

시계열 데이터를 세 가지 성분으로 분해:

```
원본 데이터 = 추세(Trend) × 계절성(Seasonal) × 잔차(Residual)
```

| 성분 | 설명 | 인사이트 |
|------|------|----------|
| **추세 (Trend)** | 장기적 방향성 | 2016-2023 지속적 상승 추세 |
| **계절성 (Seasonal)** | 반복되는 패턴 | 대회 시즌(여름/겨울) 시청 증가 |
| **잔차 (Residual)** | 설명되지 않는 변동 | 2020-2021 COVID-19 영향 포착 |

**주요 발견**:
- 2020년: COVID-19로 인한 급격한 시청 증가 (잔차에서 확인)
- 계절성: 주요 e스포츠 대회 시즌과 일치하는 패턴

## 8. 시각화 기법 및 해석

### 8.1 분포 시각화

#### 히스토그램 (Histogram)
- **목적**: 데이터의 분포 형태 파악
- **해석**:
  - 정규분포 형태 → 대칭적 분포
  - 우측 꼬리가 긴 형태 → 양의 왜도, 소수의 극단값 존재
  - 로그 스케일 적용 시 정규분포 근사 여부 확인

#### 박스플롯 (Box Plot)
- **목적**: 중앙값, 사분위수, 이상치 확인
- **해석**:
  - 상자: 25~75 백분위수 (IQR)
  - 상자 내 선: 중앙값
  - 수염: 1.5×IQR 범위
  - 점: 이상치 (outliers)

#### 바이올린 플롯 (Violin Plot)
- **목적**: 박스플롯 + 밀도 분포 결합
- **해석**:
  - 폭이 넓은 부분: 데이터가 집중된 구간
  - 비대칭 형태: 분포의 비대칭성 시각화
  - 다봉 분포(여러 봉우리): 하위 그룹 존재 가능성

#### QQ 플롯 (Q-Q Plot)
- **목적**: 정규성 시각적 검정
- **해석**:
  - 점들이 대각선 위에 있으면 → 정규분포
  - S자 형태 → 꼬리가 두꺼운 분포
  - 위로 휜 형태 → 우측 꼬리가 긴 분포

### 8.2 관계 시각화

#### 산점도 (Scatter Plot)
- **목적**: 두 변수 간 관계 파악
- **해석**:
  - 우상향 패턴 → 양의 상관관계
  - 좌상향 패턴 → 음의 상관관계
  - 무작위 패턴 → 상관관계 없음

#### 산점도 매트릭스 (Pair Plot)
- **목적**: 여러 변수 쌍의 관계를 한 번에 파악
- **해석**:
  - 대각선: 각 변수의 분포
  - 비대각선: 변수 쌍의 산점도
  - 색상: 그룹별 패턴 비교

#### 버블 차트 (Bubble Chart)
- **목적**: 3개 변수를 2D로 표현 (x, y, 크기)
- **해석**:
  - x, y 좌표: 두 변수의 관계
  - 버블 크기: 세 번째 변수의 크기
  - 큰 버블이 특정 영역에 집중 → 해당 특성의 게임이 우세

#### 히트맵 (Heatmap)
- **목적**: 행렬 데이터의 패턴 시각화
- **해석**:
  - 색상 강도: 값의 크기
  - 상관관계 히트맵: 빨강(양의 상관), 파랑(음의 상관)
  - 대각선 패턴: 자기 상관(항상 1)

### 8.3 비교 시각화

#### 막대 차트 (Bar Chart)
- **목적**: 범주별 값 비교
- **해석**:
  - 막대 높이: 해당 범주의 값
  - 정렬된 막대: 순위 파악 용이
  - 색상 구분: 그룹별 비교

#### 로리팝 차트 (Lollipop Chart)
- **목적**: 막대 차트의 세련된 대안
- **해석**:
  - 점: 정확한 값
  - 선: 기준점(0)과의 거리
  - 막대 차트보다 시각적 혼잡함 감소

#### 그룹 막대 차트 (Grouped Bar)
- **목적**: 두 그룹의 범주별 비교
- **해석**:
  - 나란히 배치된 막대로 직접 비교
  - 패턴 차이 파악 (예: 연령대별 e스포츠 vs 전통 스포츠)

### 8.4 구성 시각화

#### 파이 차트 (Pie Chart)
- **목적**: 전체 대비 부분의 비율
- **해석**:
  - 조각 크기: 해당 항목의 비율
  - 5개 이하 범주에 적합
  - 정확한 비교가 어려움 (각도 인식의 한계)

#### 도넛 차트 (Donut Chart)
- **목적**: 파이 차트의 변형, 중앙에 정보 추가 가능
- **해석**:
  - 파이 차트와 동일하나 중앙 공간 활용
  - 총합이나 핵심 지표를 중앙에 표시

#### 트리맵 (Treemap)
- **목적**: 계층적 데이터의 비율 시각화
- **해석**:
  - 사각형 크기: 해당 항목의 값
  - 공간 효율적, 많은 범주 표현 가능
  - 상위 항목이 큰 면적 차지

#### 스택 영역 차트 (Stacked Area)
- **목적**: 시간에 따른 구성 변화
- **해석**:
  - 전체 높이: 총합의 변화
  - 각 색상 영역: 해당 항목의 추이
  - 영역 넓이 변화: 점유율 변화

### 8.5 고급 시각화

#### 레이더 차트 (Radar/Spider Chart)
- **목적**: 다차원 데이터의 프로필 비교
- **해석**:
  - 각 축: 평가 항목
  - 도형 면적: 전체적인 성과
  - 형태: 강점/약점 패턴
  - 중심에 가까울수록 낮은 값

#### 3D 산점도 (3D Scatter)
- **목적**: 3개 변수의 관계를 입체적으로 표현
- **해석**:
  - x, y, z 축: 각 변수
  - 클러스터 형성: 유사한 특성의 그룹
  - 인터랙티브 회전으로 다양한 각도 확인

#### 시계열 분해 차트
- **목적**: 시계열의 구성 요소 분리
- **해석**:
  - 추세: 장기적 방향
  - 계절성: 반복 패턴
  - 잔차: 불규칙 변동 (이벤트 영향)

### 8.6 인터랙티브 시각화 (Plotly)

#### 인터랙티브 라인 차트
- **기능**: 확대/축소, 호버 정보, 범례 토글
- **장점**: 세부 데이터 탐색 가능

#### 인터랙티브 3D 차트
- **기능**: 360도 회전, 줌, 특정 점 선택
- **장점**: 복잡한 관계를 다양한 각도에서 탐색
