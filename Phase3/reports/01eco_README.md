## 1. 사용 데이터셋

### 1.1 데이터 출처 및 규모

| 데이터셋 | 레코드 수 | 출처 | 용도 |
|----------|-----------|------|------|
| e스포츠 선수 상금 | 1,000명 | Kaggle - eSports Earnings | 선수별 누적 상금 분석 |
| e스포츠 팀 데이터 | 928개 | Kaggle - eSports Earnings | 팀 단위 수입 분석 |
| e스포츠 게임별 통계 | 669개 | Kaggle - GeneralEsportData | 게임별 상금 분포 |
| e스포츠 역사 데이터 | 10,239건 | Kaggle - HistoricalEsportData | 연도별 추이 분석 |
| 글로벌 e스포츠 시장 | 400건 | Kaggle - global_gaming_esports | 시장 규모 분석 |
| FIFA 선수 데이터 | 18,207명 | Kaggle - fifa_eda_stats | 축구 선수 연봉 비교 |
| 축구 연봉 데이터 | 5,523건 | Kaggle - football_salaries | NFL 연봉 비교 |
| 올림픽 선수 데이터 | 271,116건 | Kaggle - 120 Years of Olympic | 피크 연령 분석 |

### 1.2 데이터 전처리

- **e스포츠**: 상위 100명 선수 필터링 (누적 상금 기준)
- **축구(FIFA)**: 시장가치 상위 100명 필터링, 주급 → 연봉 환산 (×52×1.08)
- **NFL**: 연봉 상위 100명 필터링
- **올림픽**: 2000년 이후 데이터만 사용 (피크 연령 분석)

### 01. 종목별 상위 선수 수입 비교

![01_top_players_comparison](output_final/01_top_players_comparison.png)

#### 시각화 유형
- 수평 막대 그래프 (Horizontal Bar Chart) × 3

#### 사용 이유
- 각 종목의 **최상위 선수들의 수입 규모**를 직관적으로 비교
- 개별 선수 이름과 금액을 함께 표시하여 구체적인 비교 가능
- 수평 막대는 긴 선수 이름을 표시하기에 적합

#### 사용 데이터
| 종목 | 데이터 | 필드 |
|------|--------|------|
| e스포츠 | esports_players_raw | TotalUSDPrize (누적 상금) |
| 축구 | fifa_players_raw | Wage × 52 × 1.08 (연간 수입) |
| NFL | football_salaries_raw | avg_year (연봉) |

#### 해석 및 결론
- **e스포츠 최고 상금**: ~$7M (Johan "N0tail" Sundstein)
- **축구 최고 연봉**: ~$30M (L. Messi, C. Ronaldo 등)
- **NFL 최고 연봉**: ~$52.5M (P. Mahomes)
- **결론**: 개별 수입 규모에서 e스포츠는 전통 스포츠 대비 15~20% 수준이나, 이는 산업 성숙도 차이를 반영

![02_income_distribution](output_final/02_income_distribution.png)

#### 시각화 유형

- 단순 평균이 아닌 **분포의 형태**를 파악
- 수입 불균형 정도를 시각적으로 확인

#### 사용 데이터
| 종목 | 샘플 수 | 데이터 필드 |
|------|---------|-------------|
| e스포츠 | 100명 | Earnings_USD |
| 축구 | 100명 | Annual_Income_USD |
| NFL | 100명 | Annual_Salary_USD |

#### 해석 및 결론
- **e스포츠**: 중앙값 ~$2M, 상위 이상치 존재 (Dota 2 선수들)
- **축구**: 중앙값 ~$8M, 비교적 균일한 분포
- **NFL**: 중앙값 ~$18M, 가장 높은 평균 및 중앙값
- **결론**: e스포츠는 전통 스포츠 대비 절대 금액은 낮으나, 분포 형태(불균형)는 유사

### 07. 매출 예측 (회귀분석)

![07_revenue_prediction](output_final/07_revenue_prediction.png)

#### 시각화 유형
- 산점도 + 회귀선 + 예측값

#### 사용 이유
- **OLS 회귀분석**을 통한 e스포츠 매출 예측
- 통계적 근거에 기반한 미래 전망 제시

#### 사용 데이터
- 독립변수: Year (2010-2023)
- 종속변수: Esports_Revenue_MillionUSD

#### 통계 결과
| 지표 | 값 |
|------|-----|
| R² | 0.8499 |
| F-statistic | 73.66 |
| p-value | < 0.001 |

#### 예측 결과
| 연도 | 예측 매출 |
|------|-----------|
| 2025 | $205.7B |
| 2026 | $220.4B |
| 2027 | $235.2B |
| 2028 | $249.9B |

- **결론**: R²=0.85로 높은 설명력, 연간 약 $15B씩 성장 예측

### 08. 로렌츠 곡선

![08_lorenz_curve](output_final/08_lorenz_curve.png)

#### 시각화 유형
- 로렌츠 곡선 (Lorenz Curve)

#### 사용 이유
- **소득 불균형 구조**를 정량적으로 비교
- "Winner-Takes-All" 가설 검증
- 전통 스포츠와 경제 구조의 동질성 확인

#### 사용 데이터
| 종목 | 데이터 | 샘플 수 |
|------|--------|---------|
| e스포츠 | esports_players_raw.TotalUSDPrize | 1,000명 |
| 축구 | fifa_players_raw.Value_USD | 5,000명 |
| NFL | football_salaries_raw.total_value | 5,523명 |

#### 지니 계수 결과
| 종목 | 지니 계수 | 해석 |
|------|-----------|------|
| e스포츠 | 0.616 | 높은 불균형 |
| 축구 | 0.487 | 중간 불균형 |
| NFL | 0.704 | 매우 높은 불균형 |

#### 결론
- e스포츠의 지니 계수(0.616)는 NFL(0.704)과 축구(0.487) 사이
- **전통 스포츠와 동일한 "Winner-Takes-All" 경제 구조** 확인
- 이는 e스포츠가 경제적으로 전통 스포츠와 유사한 산업 구조를 가짐을 의미

### 09. 인지적 부하 분석

![09_cognitive_load](output_final/09_cognitive_load.png)

#### 시각화 유형
- 불렛 차트 (Bullet Chart)

#### 사용 이유
- **프로게이머의 신경-근육 협응 능력** 정량화
- 일반인 대비 프로게이머의 능력 차이 시각화
- e스포츠가 "신체 활동"을 요구함을 입증

#### 사용 데이터 (참조 기반)
| 지표 | 프로게이머 | 일반인 | 배수 |
|------|------------|--------|------|
| APM (분당 작업 수) | 400 | 60 | 6.7배 |
| 반응 속도 (ms) | 150 | 250 | 1.7배 |
| 마우스 정확도 (%) | 98 | 85 | 1.2배 |
| 동시 정보 처리 | 12 | 7 | 1.7배 |

#### 결론
- 프로게이머는 일반인 대비 **3~6배 높은 인지적 부하** 처리
- 이는 **체조, 사격 등 초정밀 스포츠와 동등한 신체적 요구**
- e스포츠가 단순 "게임"이 아닌 "운동 능력"을 요구함을 입증
