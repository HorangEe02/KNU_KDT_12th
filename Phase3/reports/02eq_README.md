## 1. 분석 개요

### 1.1 분석 목적

본 분석은 **e스포츠가 전통 스포츠와 경제적으로 동등한 지위를 갖는지**를 데이터 기반으로 증명하는 것을 목적으로 합니다.

### 1.2 분석 방법론

- **비교 분석**: e스포츠와 전통 스포츠(NFL, 축구, 테니스, 골프 등) 간 다차원 비교
- **통계적 분석**: 지니 계수, CAGR, 상관관계 분석
- **시각화**: 14개의 차트를 통한 데이터 시각화

### 1.3 분석 기간

- 데이터 기간: 1998년 ~ 2024년
- 분석 수행일: 2025년 1월

#### 시각화 구성

- **좌측 차트**: 전체 10개 종목의 시장 규모 비교 (수평 막대 그래프)
- **우측 차트**: e스포츠 vs 개별/올림픽 종목 비교 (동일 스케일)

#### 데이터 값

```
축구(FIFA): $280B    |  NFL: $180B         |  NBA: $100B
골프: $100B          |  테니스: $60B       |  e스포츠: $18B

```

#### 해석

1. **메이저 스포츠 vs e스포츠**: 축구($280B), NFL($180B), NBA($100B) 등 메이저 스포츠는 e스포츠($18B)보다 10~15배 큰 시장 규모를 보유
2. **e스포츠의 상대적 위치**: e스포츠 시장($18B)은 UFC($12B)보다 크고, 테니스($60B)의 약 30% 수준
3. **올림픽 종목과의 비교**: e스포츠는 양궁($2B), 펜싱($1.5B), 사격($3B) 등 대부분의 올림픽 종목보다 **6~12배 큰 시장**

#### 결론

> **e스포츠는 메이저 스포츠와는 격차가 있으나, 대다수의 올림픽 정식 종목보다 경제적 규모가 크다.**
> 이는 e스포츠가 "스포츠"로서의 경제적 기반을 충분히 갖추고 있음을 시사한다.

3. **전통 스포츠와의 유사점**: 수익 구조의 패턴이 전통 스포츠와 매우 유사

> **e스포츠의 수익 구조(미디어+스폰서십 65%)는 전통 스포츠와 동일한 비즈니스 모델을 따른다.**
> 이는 e스포츠가 전통 스포츠와 동일한 산업 구조를 갖추고 있음을 증명한다.

### 3.3 로렌츠 곡선 및 지니 계수

![로렌츠 곡선](final_output/03_lorenz_curve.png)

#### 데이터 출처

| 종목        | 데이터 출처               | 변수                      |
| ----------- | ------------------------- | ------------------------- |
| e스포츠     | `GeneralEsportData.csv` | `TotalEarnings`         |
| 축구 (FIFA) | `fifa_eda_stats.csv`    | `Value` (선수 시장가치) |
| NFL         | `football_salaries.csv` | `avg_year` (연봉)       |

#### 선택 변수

| 변수명             | 설명                                          | 계산 방법                    |
| ------------------ | --------------------------------------------- | ---------------------------- |
| `인구 누적 비율` | 선수/팀을 소득 순으로 정렬 후 누적 비율       | `cumsum(rank) / n`         |
| `소득 누적 비율` | 해당 비율까지의 소득 누적 합                  | `cumsum(earnings) / total` |
| `지니 계수`      | 소득 불균형 지수 (0=완전 평등, 1=완전 불평등) | Gini formula                 |

#### 시각화 구성

- **로렌츠 곡선**: 3개 종목의 소득 분포 곡선
- **대각선**: 완전 평등선 (45도 직선)
- **면적**: 로렌츠 곡선과 대각선 사이 면적 = 불평등 정도

#### 계산된 지니 계수

| 종목              | 지니 계수       | 해석                         |
| ----------------- | --------------- | ---------------------------- |
| **e스포츠** | **0.937** | 매우 높은 불평등 (상위 집중) |
| 축구 (FIFA)       | 0.715           | 높은 불평등                  |
| NFL               | 0.607           | 중간~높은 불평등             |

#### 해석

1. **승자독식 구조**: e스포츠(0.937)는 전통 스포츠보다 더 극심한 소득 불균형
   - 상위 소수 선수/팀이 전체 상금의 대부분을 가져감
   - 이는 토너먼트 기반 상금 구조의 특성
2. **전통 스포츠와의 유사성**:
   - 축구(0.715), NFL(0.607)도 높은 지니 계수
   - 프로 스포츠 산업의 본질적 특성 (스타 플레이어 프리미엄)
3. **경제 구조 동질성**: 모든 프로 스포츠는 "승자독식" 구조를 공유

#### 결론

> **e스포츠의 소득 분배 구조(지니 계수 0.937)는 전통 스포츠와 동일한 "승자독식" 패턴을 보인다.**
> 높은 지니 계수는 프로 스포츠 산업의 보편적 특성이며, e스포츠도 동일한 경제 법칙을 따른다.

### 3.4 ARPU 버블 차트

![ARPU 버블 차트](final_output/04_arpu_bubble.png)

#### 데이터 출처

- **e스포츠**: Newzoo Esports Market Report 2020-2024
- **NFL**: NFL Revenue Reports, Statista
- **축구**: Deloitte Football Money League

#### 선택 변수

| 변수명              | 설명               | 단위       |
| ------------------- | ------------------ | ---------- |
| `viewers_million` | 시청자 수          | 백만 명    |

#### 시각화 구성

- **X축**: 시청자 수 (백만 명)
- **Y축**: 매출액 (십억 달러)
- **버블 크기**: 성장률
- **색상**: 종목별 구분

#### 데이터 값 (2024년 기준)

| 종목    | 시청자(M) | 매출(B)         | ARPU  | 성장률 |
| ------- | --------- | --------------- | ----- | ------ |
| e스포츠 | 800       | $1.87 | $2.34   | 37.3% |        |
| NFL     | 170       | $18.0 | $105.88 | 6.0%  |        |
| 축구    | 4,000     | $35.0 | $8.75   | 4.2%  |        |

#### 해석

1. **ARPU 비교**:
   - NFL: $105.88/viewer (가장 높음)
   - 축구: $8.75/viewer
   - e스포츠: $2.34/viewer (가장 낮음)
2. **성장 잠재력**:
   - e스포츠의 낮은 ARPU는 **수익화 여지가 크다**는 것을 의미
   - 37.3%의 높은 성장률이 이를 반증
3. **시청자 규모**: e스포츠(8억)는 NFL(1.7억)보다 많지만 수익화 효율이 낮음

#### 결론

> **e스포츠는 시청자당 수익(ARPU)이 낮지만, 이는 시장 성숙도가 낮아 수익화 잠재력이 크다는 것을 의미한다.**
> 높은 성장률(37.3%)은 향후 ARPU 개선 가능성을 시사한다.

### 3.5 인지적 부하 불렛 차트

![인지적 부하](final_output/05_cognitive_load.png)

#### 데이터 출처

- **APM 데이터**: 학술 연구 (Starcraft II Pro Player Analysis)
- **반응 속도**: Human Benchmark, 스포츠 과학 연구
- **정밀도**: e스포츠 선수 퍼포먼스 데이터
- **동시 정보 처리**: 인지 심리학 연구

#### 선택 변수

| 변수명             | e스포츠 선수 | 일반인/전통 스포츠 | 단위        |
| ------------------ | ------------ | ------------------ | ----------- |
| APM (분당 작업 수) | 400          | 60                 | Actions/min |
| 반응 속도          | 150          | 250                | ms          |
| 정밀도             | 98           | 85                 | %           |
| 동시 정보 처리     | 12           | 7                  | objects     |

#### 시각화 구성

- **불렛 차트**: 4개 인지 지표의 성능 비교
- **막대**: e스포츠 선수 수치
- **빨간 선**: 일반인/전통 스포츠 기준점
- **배경**: 성능 범위 구간

#### 해석

1. **APM (400 vs 60)**:
   - 프로게이머는 일반인보다 **6.7배 빠른** 작업 처리
   - 이는 피아니스트의 손가락 움직임에 필적
2. **반응 속도 (150ms vs 250ms)**:
   - 40% 더 빠른 반응 속도
   - F1 드라이버(150ms), 야구 타자(150ms)와 유사
3. **정밀도 (98% vs 85%)**:
   - 사격 선수급의 정밀 제어 능력
4. **동시 정보 처리 (12 vs 7)**:
   - 평균 인간(7개)보다 71% 더 많은 객체 동시 추적
   - 항공 관제사 수준의 멀티태스킹

#### 결론

> **프로게이머는 신경-근육 협응, 반응 속도, 정보 처리 능력에서 전문 운동선수에 필적하는 수준을 보인다.**
> 이는 e스포츠가 "정신적 스포츠"가 아닌 "신체적 기술을 요구하는 스포츠"임을 증명한다.

   - 반응 속도, 정밀 제어가 중요한 종목의 특성
2. **종목별 군집화**:
   - **순발력 스포츠** (피크 20-22세): 체조, 수영, e스포츠
   - **지구력 스포츠** (피크 25-28세): 축구, 농구
   - **기술/경험 스포츠** (피크 30-35세): 골프, 사격
3. **빠른 은퇴 연령**: e스포츠 선수들은 25세 이후 경쟁력 급감

#### 결론

> **e스포츠의 피크 연령(22세)은 체조, 수영과 같은 "초정밀 순발력 요구 스포츠" 군집에 속한다.**
