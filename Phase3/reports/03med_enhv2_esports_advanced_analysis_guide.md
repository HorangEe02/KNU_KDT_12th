### 과제 1: 신체적 부하의 재정의 — 불렛 차트(Bullet Chart)

#### 1.1 분석 배경

현재 분석에서 '신체/인지적 요구' 점수가 40점으로 낮게 책정된 이유는 "e스포츠 선수는 앉아서 게임만 한다"는 고정관념 때문입니다. 그러나 경기 중 심박수, 반응 속도, 코르티솔 수치 등을 살펴보면 e스포츠 선수들이 겪는 생리적 스트레스는 전통 스포츠 선수와 비견될 수 있습니다.

#### 1.2 필요 데이터

#### 1.3 시각화 방법: 불렛 차트(Bullet Chart)

불렛 차트는 실제 값(Actual)을 목표 값(Target) 및 배경 범위(Qualitative Ranges)와 비교하는 데 효과적입니다.

```
시각화 구성 요소:
┌─────────────────────────────────────────────────────────────┐
│  배경 영역 (3단계 색상 구분)                                    │
│  ├─ 회색 (60-80 bpm): 안정시 심박수 (Resting)                  │
│  ├─ 연두색 (80-140 bpm): 유산소 운동 구간 (Aerobic Zone)        │
│  └─ 주황색 (140-180 bpm): 고강도 운동 구간 (Anaerobic Zone)     │
│                                                             │
│  막대 (Actual Bar): e스포츠 선수의 경기 중 평균 심박수           │
│  기준선 (Target Line): 사격/양궁/바둑 선수의 평균 심박수         │
└─────────────────────────────────────────────────────────────┘
```

### 과제 2: 성공 메커니즘의 동질성 — 이중 회귀선 산점도

#### 2.2 필요 데이터

기존 데이터셋에서 다음 변수를 활용하세요.

|------|--------------------------------|----------------------|
| CS:GO | Maps_Played (경기 수) | Rating, KD_Ratio |
| FIFA/축구 | Age - 17 (추정 경력 연수) | Overall, Value |
| NFL | Age - 22 (추정 경력 연수) | (가능하면 연봉 또는 성과 지표) |

### 과제 4: 포지션별 분화의 정교화 — 레이더 차트(Radar Chart)

### 보강 과제 A: 효과 크기(Effect Size) 시각화

#### A.1 현재 문제점

기존 분석에서 t-test 결과의 p-value만 제시하고 있습니다. 그러나 p-value는 "차이가 있는가?"만 알려줄 뿐, "얼마나 큰 차이인가?"는 알려주지 않습니다.

#### A.2 보강 방법

```python
# Cohen's d 값을 시각화에 포함

# 1. 기존 분석에서 계산한 Cohen's d 값 활용
# 2. 막대그래프로 효과 크기 비교 시각화

# Cohen's d 해석 기준
# |d| < 0.2  : 작은 효과 (Small)
# |d| 0.2-0.8: 중간 효과 (Medium)
# |d| > 0.8  : 큰 효과 (Large)

# 시각화 예시: 수평 막대그래프
# X축: Cohen's d 값
# Y축: 비교 항목 (e스포츠 vs 축구, e스포츠 vs NFL 등)
# 배경: 효과 크기 구간별 색상 음영
```

#### A.3 시각화 구성

```

│                                                             │

│  ───────────────────────────────────────────────            │
│                    Small   Medium    Large                  │
│                    |0.2|    |0.5|    |0.8|                  │
│                                                             │
│  연령: e스포츠 vs 축구    ████░░░░░░░░░░░  d = 0.18         │
│  연령: e스포츠 vs NFL     ████████████░░░  d = 0.72         │
│  Rating vs Overall       ██████░░░░░░░░░  d = 0.35         │
│                                                             │
│  해석: "통계적으로 유의미하지만, 실질적 차이는 중간 수준"      │

```

### 보강 과제 B: APM을 신체 활동량의 대리 변수로 활용

#### B.1 분석 배경

README에서 "e스포츠 선수의 신체 데이터 부재"를 한계점으로 인정했습니다. 이를 창의적으로 해결하기 위해 **APM(Actions Per Minute)**을 '신체 활동량'의 대리 변수(Proxy Variable)로 설정합니다.

#### B.2 논리적 근거

APM이 높다는 것은 다음을 의미합니다.
1. 손가락과 손목의 빈번한 움직임 (미세 근육 사용)
2. 높은 인지 부하 (빠른 의사결정)
3. 지속적인 집중력 유지 (정신적 체력 소모)

이는 전통 스포츠에서 "분당 달린 거리"나 "분당 동작 수"와 유사한 개념입니다.

#### B.3 필요 데이터

웹 검색을 통해 다음 데이터를 수집하세요.

| 게임 | 프로 선수 평균 APM | 일반 유저 평균 APM |
|------|-------------------|-------------------|
| StarCraft II | 300-400+ | 50-100 |
| LoL | 200-300 | 30-60 |
| Dota2 | 150-250 | 40-80 |
| CS:GO | (마우스 움직임/분) | - |

#### B.4 시각화 방법

```python
# 1. APM 분포 비교: 프로 vs 일반 유저
#    - 히스토그램 또는 박스플롯

# 2. APM과 성과의 상관관계
#    - 산점도 + 회귀선
#    - "APM이 높을수록 성과가 좋다"는 관계 입증

# 3. 전통 스포츠의 유사 지표와 비교
#    - 축구: 분당 스프린트 횟수
#    - 농구: 분당 동작 수
#    - 정규화하여 비교
```

#### B.5 전달해야 할 메시지

> "e스포츠 프로 선수의 APM은 일반인의 4-6배에 달하며, 이는 경기 중 손과 손가락이 쉬지 않고 움직인다는 것을 의미한다. 신체 전체를 사용하지 않을 뿐, 미세 운동 능력의 요구 수준은 전통 스포츠의 그것과 다르지 않다."
