## 2. 척도별 MBTI 분류 방법

### 2.1 공통 원리

- **7점 리커트/양극 척도**: 1~7 범위
- **중립점(midpoint)**: 4.0 (이론적 중앙값)
- **분류 기준**: 각 차원의 평균 점수를 기준으로
  - **>= 4.0** -> 첫 번째 글자 (E / S / T / J)
  - **< 4.0** -> 두 번째 글자 (I / N / F / P)
- **최종 MBTI**: 4개 차원의 글자를 결합 (예: E+N+F+P = "ENFP")

### 2.2 선호 강도 (Preference Clarity Index)

중립점(4.0)으로부터의 거리로 선호 강도를 판정합니다.

| 거리 (|d|) | 강도 | 의미 |
|-----------|------|------|
| 0.00 ~ 0.74 | 약간 (Slight) | 경계선에 가까움, 상황에 따라 변할 수 있음 |
| 0.75 ~ 1.49 | 보통 (Moderate) | 한쪽으로 기울어진 선호 |
| 1.50 ~ 2.24 | 명확 (Clear) | 뚜렷한 성향 |
| 2.25 이상 | 매우 명확 (Very Clear) | 강한 성향 |

## 3. 차원별 채점 상세

### 3.1 E/I 차원 — "인싸인가 아싸인가"

**특징**: **단일 방향** 문항 (전체 9문항 I방향 — 높은 점수 = 내향)

#### 문항 구성

| Q번호 | 방향 | 원점수 높으면? | 채점 | 문항 내용 |
|-------|------|-------------|------|----------|
| Q6 | I | 내향(I) | **역채점** | 주말을 침대에서 보내고 싶다 |
| Q7 | I | 내향(I) | **역채점** | 약속 취소 = 은근히 좋다 |
| Q8 | I | 내향(I) | **역채점** | 주중에는 회사(실외)였으니 주말엔 집에 있어야 한다 |
| Q9 | I | 내향(I) | **역채점** | 놀고 나면 집에서 충전 |
| Q10 | I | 내향(I) | **역채점** | 몇 개월 집밖 안 나가도 OK |
| Q11 | I | 내향(I) | **역채점** | 당일 갑자기 잡히는 약속이 힘들다 |
| Q12 | I | 내향(I) | **역채점** | 볼일 하루에 몰아서 끝냄 |
| Q13 | I | 내향(I) | **역채점** | 새 모임에서 빨리 가고 싶다 |
| Q14 | I | 내향(I) | **역채점** | 빈 강의실에 벽쪽 앉음 |

#### 채점 과정

```
1. 전체 9문항(Q6~Q14) 역채점 (8 - 원점수)
2. 9개 문항의 평균 계산
3. 평균 >= 4.0 → E (외향), 평균 < 4.0 → I (내향)
```

#### 예시

| 문항 | 원점수 | 채점 방향 | 채점 결과 |
|------|--------|----------|-----------|
| Q6 (I) | 6 | 역채점 | 8-6 = **2** |
| Q7 (I) | 5 | 역채점 | 8-5 = **3** |
| Q8 (I) | 6 | 역채점 | 8-6 = **2** |
| Q9 (I) | 6 | 역채점 | 8-6 = **2** |
| Q10 (I) | 7 | 역채점 | 8-7 = **1** |
| Q11 (I) | 5 | 역채점 | 8-5 = **3** |
| Q12 (I) | 5 | 역채점 | 8-5 = **3** |
| Q13 (I) | 6 | 역채점 | 8-6 = **2** |
| Q14 (I) | 5 | 역채점 | 8-5 = **3** |
| **평균** | | | **(2+3+2+2+1+3+3+2+3)/9 = 2.33** |

결과: 2.33 < 4.0 -> **I (내향)**
거리: |4.0 - 2.33| = 1.67 -> 명확 (Clear)

> 이 응답자는 전형적인 "집순이/집돌이" 성향을 보입니다.

### 3.2 S/N 차원 — "현실주의자인가 몽상가인가"

**특징**: **단일 방향** 양극 척도 (1=S, 7=N)

#### 문항 구성

| Q번호 | 원래 척도 | 채점 | 문항 키워드 |
|-------|----------|------|-----------|
| Q15 | 1=현실 ↔ 7=상상 | **역채점** | 과제 깜빡: 현실적 반응 vs 극단적 상상 |
| Q16 | 1=현실 ↔ 7=상상 | **역채점** | 축제: 눈앞 무대 vs 내가 참가 상상 |
| Q17 | 1=현실 ↔ 7=상상 | **역채점** | 망상 스타일: 현실 기반 vs 판타지급 |
| Q18 | 1=현실 ↔ 7=상상 | **역채점** | 드라마: 고증 체크 vs 2차 창작 |
| Q19 | 1=현실 ↔ 7=상상 | **역채점** | 성과 부진: 원인 분석 vs 파국적 상상 |
| Q20 | 1=현실 ↔ 7=상상 | **역채점** | 버스 창밖: 현실 관찰 vs 세계 상상 |
| Q21 | 1=현실 ↔ 7=상상 | **역채점** | "사과 하면?" 감각 vs 연상 |
| Q22 | 1=현실 ↔ 7=상상 | **역채점** | 시험: 해결책 vs 이상 세계 |
| Q23 | 1=현실 ↔ 7=상상 | **역채점** | 소풍 전날: 체크리스트 vs 시나리오 |

#### 채점 과정

```
1. 모든 9개 문항: 역채점 (8 - 원점수)
   → 원래 1=S, 7=N 이었던 척도가 → 1=N, 7=S 로 변환
   → 높은 점수 = S(감각) 성향
2. 9개 문항의 평균 계산
3. 평균 >= 4.0 → S (감각), 평균 < 4.0 → N (직관)
```

> **왜 역채점?**
> v1 채점 시스템과 동일한 컨벤션(높은 점수 = 첫 번째 글자)을 유지하기 위해서입니다.
> 원점수가 높으면 N이지만, 역채점하면 높은 점수 = S가 됩니다.

## 8. 주의사항

1. **역채점 일관성**: v2의 S/N, T/F, J/P는 양극 척도이므로 전체 역채점이 필요합니다.
   역채점을 빠뜨리면 E/S/T/J ↔ I/N/F/P가 뒤바뀝니다.

2. **E/I 단일 방향**: E/I도 S/N, T/F, J/P와 동일하게 전체 9문항이 I방향이므로,
   전부 역채점(8-원점수)해야 합니다. 4개 차원 모두 동일한 채점 방식입니다.

3. **v1과 호환**: 역채점 후 최종 점수 컨벤션은 v1과 동일하므로,
