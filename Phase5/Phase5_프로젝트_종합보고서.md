# 기(起) — 프로젝트 배경과 동기

## 1. 왜 이 프로젝트를 수행하게 되었는가

### 1.1 대구광역시의 위기 현실

대구광역시는 현재 복합적인 도시 위기에 직면해 있다. 이 프로젝트는 대구가 처한 위기의 심각성을 데이터로 확인하고, 유사한 도전을 극복한 해외 도시 사례에서 해법을 찾기 위해 시작되었다.

**인구 위기:**
- 대구 인구: 249만 명(2015) → 236만 명(2023)으로 감소 가속화
- 20대 청년층: 31.5만 명 → 25.3만 명으로 급감 (대구일보 보도: "20대 청년층 급감")
- 2019년 인구 데드크로스 발생: 출생(13,233명) < 사망(13,752명), 자연증감 -519명
- 2022년 자연감소 가속: 출생 5,256명 vs 사망 9,259명 = **자연증감 -4,003명**
- 출생률 22년간 83.8% 급감 (32,477명 → 5,256명)
- 혼인율 50% 하락 (6.4‰ → 3.1‰) — 출생률 추가 하락의 선행 지표

**경제 구조 문제:**
- 대기업 부재로 인한 고소득 일자리 부족
- 섬유·자동차 부품 중심의 전통 산업 구조
- 한국 제조업 의존도 27.9%로 미국(10.7%)의 2.6배
- 청년층이 수도권으로 유출되는 악순환

**부동산 시장 교착:**
- 2020~2021년 저금리+투자 수요로 가격 급등 (평균 거래가 5,000만원 → 35,000만원)
- 2022년 이후 거래건수 급감 (63,000건 → 18,000건), 가격은 하방경직
- 인구 감소에도 가격이 유지되는 비정상적 "교착 상태"

### 1.2 프로젝트 핵심 질문

> **"인구가 줄어드는 도시의 부동산 시장은 어떻게 될 것인가? 유사한 위기를 극복한 해외 도시에서 해법을 찾을 수 있는가?"**

이 질문에 답하기 위해, 대구와 기후·산업·역사적으로 유사한 미국 4개 내륙 거점 도시를 선정하여 데이터 기반 비교 분석을 수행하였다.

### 1.3 비교 도시 선정 근거

| 비교 축 | 대구광역시 | 미국 유사 도시 | 선정 이유 |
| ------- | ---------- | -------------- | --------- |
| **종합** | 경상도 거점 대도시 | **Dallas, TX** | 내륙 대도시, 보수적 성향, 무더운 여름, 교통 요충지 |
| **산업** | 자동차 부품, 에너지 | **Atlanta, GA** | 자동차/배터리 허브, 교통 요충지, IT·물류 성장 |
| **기후** | 분지 지형, 폭염 | **Phoenix, AZ** | 극심한 폭염, 열섬 현상, 태양광/에너지 연구 활발 |
| **변모** | 섬유 → 첨단 산업 | **Charlotte, NC** | 섬유 산업의 역사 → 금융/에너지/첨단 도시로 전환 |

### 1.4 팀 구성원의 전공 연결

질문자(팀원)가 **통계학 전공 + 자동차 공학/수소 에너지 복수 전공** 과정에 있어, 데이터 분석 역량과 자동차·에너지 산업 지식을 프로젝트에 직접 활용할 수 있었다. 특히 Atlanta(자동차/배터리 허브)와 Phoenix(신재생 에너지)는 전공과 직결되는 도시 선정이었다.


### 3.2 Subtopic 2: 인구-수요 관계 — 핵심 발견

#### 가장 중요한 발견: "대구 주택가격의 핵심 동인은 인구가 아니라 소득"

피어슨 상관분석 결과:

| 독립변수 (X) | 종속변수 (Y) | 상관계수 (r) | p-value | 유의성 |
| ------------ | ------------ | ----------- | ------- | ------ |
| **중위소득** | **ZHVI** | **+0.845** | **0.00014** | *** |
| 인구변동률 | ZHVI변동률 | +0.034 | 0.896 | n.s. |
| 인구변동률 | 판매건수 | +0.017 | 0.946 | n.s. |

→ **소득과 주택가격: r=0.845 (매우 강한 상관)**
→ **인구와 주택가격: r=0.034 (사실상 무상관)**
→ "인구만 늘리면 집값이 오른다"는 단순 논리는 대구에 적용되지 않음

#### 인구유출+가격상승의 역설 (2019~2021)

| 기간 | 패턴 | 횟수 | 대표 사례 |
| ---- | ---- | ---- | --------- |

| 2008-2018 | 인구유입+가격상승 | 11회 | 정상적 상관 |

### 3.3 Subtopic 3: 기후-가격 격차 — 핵심 발견

#### Charlotte 산업 전환 모델 = 대구의 벤치마크

- Charlotte 전환 성공 신호: 신규건설 22배 증가(800건→18,000건) + 필요소득 $30K→$98K

#### 인구 없는 가격 상승은 지속 불가능

- **미국 4도시**: 인구 유입 → 실수요 기반 가격 상승 → 금융위기에서도 결국 회복
- **대구**: 인구 유출 속 투자 수요로 가격 급등 → 2022년 이후 거래 급감, 교착 상태
- Charlotte은 건설+가격 동반 상승("선순환"), 대구는 가격만 높고 거래 급감("투기 후 교착")

# 결(結) — 자체 평가

## 4. 프로젝트를 통해 무엇을 얻었는가

### 4.1 결과 해석

본 프로젝트는 대구광역시의 위기를 **"감정이 아닌 데이터로"** 진단하고, 미국 4개 유사 도시와의 비교를 통해 **구체적이고 정량적인 전략**을 도출하는 데 성공하였다.

**가장 의미 있는 발견 3가지:**

1. **"소득이 핵심이다" (r=0.845 vs r=0.034)**
   - 대구 주택가격을 결정하는 것은 인구가 아닌 소득이라는 통계적 증거를 확보하였다. 이는 "인구를 늘리면 집값이 오른다"는 상식적 가정을 데이터로 반증하는 결과로, 정책 방향에 중요한 시사점을 제공한다.

2. **"인구 없는 가격 상승은 지속 불가능하다"**
   - 2019~2021년 대구의 "인구유출+가격상승" 역설은 저금리와 투자 수요라는 비정상적 요인에 의한 것이었음을 확인하였다. 미국 도시들은 인구 유입이라는 실수요 기반 위에서 가격이 상승한 반면, 대구는 그렇지 못했다.

3. **"향후 5~10년이 골든타임이다"**
   - Charlotte의 30년 전환 사례에 비추어 대구도 이미 30년째 전환 중이다. 미국 4도시 시장온도가 50 전후로 수렴하며 냉각기에 진입한 지금이, 대구가 가격 경쟁력을 무기로 인구를 유치할 수 있는 전략적 기회의 창이다.

### 4.2 느낀 점

**데이터 분석의 힘:**
- 947,791건의 대구 실거래가와 미국 주요 데이터셋을 결합하여, 감이나 경험이 아닌 **통계적 근거에 기반한 인사이트**를 도출할 수 있었다. 특히 피어슨 상관분석(r=0.845, p<0.001)이 "소득이 핵심"이라는 메시지를 정량적으로 뒷받침한 것이 인상적이었다.

**SQL + Python 파이프라인의 효율성:**
- MySQL을 데이터 저장·조회의 중심으로, Python을 분석·시각화의 도구로 결합한 파이프라인이 매우 효과적이었다. `run_all.py`로 전체 분석을 자동 재현할 수 있어, 데이터 변경 시에도 즉시 결과를 갱신할 수 있는 **재현 가능한 분석 체계**를 구축했다.

**팀워크와 역할 분담:**
- 4개 소주제를 팀원별로 분담하되, 공통 DB 스키마와 도시 선정 프레임워크를 먼저 합의함으로써 일관성 있는 결과를 도출할 수 있었다. 개별 분석 결과가 종합될 때 하나의 스토리로 자연스럽게 연결되는 경험이 프로젝트 설계의 중요성을 체감하게 했다.

**실제 도시 문제와의 연결:**
- 추상적인 학습이 아니라, 본인이 살고 있는 대구라는 도시의 실제 문제를 데이터로 분석한다는 점에서 동기부여가 강했다. 특히 보육시설 35% 감소, 청년층 급감 등의 통계가 피부로 와닿는 현실이었기에 분석에 더 몰입할 수 있었다.

### 4.3 보완할 사항

**데이터 관련:**

| 한계 | 설명 | 개선 방향 |
| ---- | ---- | --------- |
| 통화 직접 비교 불가 | 대구(KRW)와 미국(USD) 절대 금액 비교 제한 | PPP(구매력평가) 기반 보정 도입 |
| 대구 임대 데이터 부재 | 전·월세 데이터 미포함으로 ZORI와 직접 비교 불가 | 국토부 전월세 실거래가 데이터 추가 |
| 인구 이동 데이터 미반영 | 선벨트 도시 가격 급등의 핵심 동인인 이주 데이터 부재 | Census 인구이동 데이터, 대구 전입/전출 통계 추가 |
| 시장건전성 데이터 기간 제한 | 가격 인하 매물 비율이 2017년까지만 존재 | 최신 Zillow API 연동 또는 대안 데이터 탐색 |
| 기후 데이터 기간 제한 | Berkeley Earth가 2013년까지만 존재 | NOAA 최신 기후 데이터로 확장 |

**분석 방법론 관련:**
- 피어슨 상관분석은 선형관계만 측정하므로, 비선형 관계나 시차 효과(lag effect)를 반영하기 어려웠다. 향후 시계열 분석(VAR, Granger 인과성 검정)으로 인과관계를 더 엄밀히 검증할 필요가 있다.
- ZIP 코드 수준의 미시 분석에서 표본 크기가 제한적(Phoenix 다발 ZIP 2개 등)인 경우가 있어, 결론의 일반화에 한계가 있었다.
- 나고야(일본) 비교 분석도 발표에서 다루었으나, 본 main_usa 프로젝트에서는 미국 도시에 집중하여 나고야 데이터의 정량적 비교가 누락되었다.

**발표·보고 관련:**
- 29개 시각화와 37개 CSV의 방대한 산출물을 효과적으로 전달하기 위한 스토리텔링 구조화가 더 필요했다.
- 발표자료(PPT)와 분석 보고서 간의 일관성 유지가 어려웠으며, 자동화된 보고서 생성 체계가 있었다면 더 효율적이었을 것이다.

### 4.4 현재 검증된 능력

본 프로젝트를 통해 다음의 역량이 실증적으로 검증되었다:

**1. 데이터베이스 설계 및 운용 능력**
- MySQL에 46개 테이블을 설계하고, 총 **약 180만 행** 이상의 데이터를 적재·관리
- ERD 설계, 테이블 간 관계 정의, 인덱스 최적화
- 복잡한 JOIN, 서브쿼리, 집계 함수를 활용한 분석 쿼리 작성

**2. Python 데이터 분석 파이프라인 구축 능력**
- PyMySQL을 통한 DB 연동, Pandas를 활용한 데이터 가공
- SciPy를 이용한 통계 분석 (피어슨 상관, p-value 검정)
- `run_all.py` 오케스트레이터로 전체 파이프라인 자동화 (에러 0건 달성)

**3. 데이터 시각화 및 해석 능력**
- Matplotlib/Seaborn으로 29개 시각화 생성 (라인차트, 히트맵, 레이더차트, 박스플롯, 인포그래픽 등)
- 각 시각화에 대해 "왜 이 차트 유형인가"에 대한 명확한 근거 제시
- 데이터를 비즈니스 인사이트로 전환하는 해석 능력

**4. 통계적 사고와 정책 제언 능력**
- 피어슨 상관분석으로 "소득 vs 인구" 가설을 검증하고, 직관에 반하는 결과(인구 무상관)를 데이터로 입증
- 미국 4도시의 실증 데이터에서 대구에 적용 가능한 전략을 도출
- 단기·중기·장기 시간축을 가진 실행 로드맵과 KPI 설정

**5. 팀 프로젝트 수행 및 문서화 능력**
- 4명이 4개 소주제를 독립적으로 분석하되 일관된 프레임워크 유지
- 분석 보고서, 발표자료(PPT), 전략 보고서 등 다양한 형태의 산출물 생성
- 전체 프로젝트의 재현 가능성 확보 (코드, 데이터, 문서 일체 보존)
