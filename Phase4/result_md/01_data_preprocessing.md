## 4. 데이터 전처리 과정

### 4.1 전처리 파이프라인

```
원본 데이터 로드 → 청크 처리 → 품목 필터링 → 피벗 변환 → 기온 데이터 병합 → 통합 데이터셋 생성
```

### 4.2 수행된 데이터 변환

#### 4.2.1 대용량 데이터 청크 처리

| 변환 내용                            | 이유                                                                          |
| ------------------------------------ | ----------------------------------------------------------------------------- |
| 청크 사이즈 500,000 rows로 분할 처리 | FAOSTAT 원본 데이터가 수백만 행의 대용량 데이터이므로 메모리 효율적 처리 필요 |
| 필요한 Item Code만 실시간 필터링     | 전체 데이터를 메모리에 로드하지 않고 필요한 품목만 추출하여 처리 속도 향상    |

```python
for chunk in pd.read_csv(production_path, chunksize=500000):
    filtered = chunk[chunk['Item Code'].isin(target_item_codes)]
```

#### 4.2.2 연도 필터링

| 변환 내용                    | 이유                                                            |
| ---------------------------- | --------------------------------------------------------------- |
| 2000년 이후 데이터만 사용    | 최근 25년간의 데이터에 집중하여 현대 기후 변화 영향 분석에 적합 |
| 기후 변화 가속화 시기와 일치 | 2000년대 이후 지구 온난화가 뚜렷하게 가속화된 시기              |

#### 4.2.3 피벗 테이블 변환

| 변환 전               | 변환 후               | 이유                                  |
| --------------------- | --------------------- | ------------------------------------- |
| Long format (행 기반) | Wide format (열 기반) | 국가별 시계열 비교 분석 용이          |
| Area가 행으로 반복    | Area가 열로 전개      | 시각화 및 상관관계 분석에 적합한 구조 |

```python
df_pivot = df_elem.pivot_table(
    index='Year',
    columns='Area',
    values='Value',
    aggfunc='sum'
)
```

#### 4.2.4 기온 데이터 집계

| 변환 내용                                   | 이유                                                           |
| ------------------------------------------- | -------------------------------------------------------------- |
| 월별 데이터 → 연간 평균                    | 생산량 데이터가 연간 단위이므로 기온 데이터도 연간 단위로 집계 |
| 국가별 기온 편차 → 전 세계 평균(World_Avg) | 전 세계적 기후 변화 추세와 생산량 관계 분석                    |

#### 4.2.5 파생 변수 생성

| 변수명             | 계산 방법                   | 이유                                |
| ------------------ | --------------------------- | ----------------------------------- |
| `World_Total`    | 주요 생산국 생산량 합계     | 전 세계 총 생산량 추세 파악         |
| `YoY_Change_Pct` | (금년 - 전년) / 전년 × 100 | 기후 변화에 따른 생산량 변동성 분석 |
| `Temp_Anomaly_C` | 기온 데이터 병합            | 생산량-기온 상관관계 분석           |
