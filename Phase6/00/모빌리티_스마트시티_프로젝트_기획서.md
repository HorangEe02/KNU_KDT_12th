## 1. 프로젝트 개요

### 대주제

> **모빌리티·스마트시티 산업에서 AI·빅데이터·Python 인재는 어떤 역량을 갖춰야 하는가?**

### 스토리라인

자율주행, 의료, 스마트팩토리, GIS, 자동차가 모두 **"스마트시티"** 라는 큰 흐름 안에서 연결되며, 각 분야에서 요구하는 인재상과 현실적 근무 환경을 종합 비교합니다.

### 4가지 분석 축

| 분석 축              | 핵심 내용                                 |
| -------------------- | ----------------------------------------- |
| **① 취업 현황 분석** | 채용 공고 수, 경력/신입 비율, 분야별 통계 |
| **② 요구 역량 분석** | 기술스택, 자격증, 우대사항                |
| **③ 준비 동향 분석** | 취준생이 준비하는 교육·자격증·도서        |
| **④ 근무 환경 분석** | 연봉, 평점, 복지, 기업 리뷰               |

## 2. 팀원별 분석 영역

### 🏭 박 건희 — 스마트팩토리·제조 기술

- **분석 목표**: 스마트팩토리·제조업에서 빅데이터/AI 직군의 채용 현황과 기존 제조 기술직과의 요구 역량 차이 분석
- **핵심 키워드**: 스마트팩토리, MES, PLC, 품질 데이터, 생산 AI
- **대상 기업(예시)**: 삼성SDS, LS산전, 포스코ICT, 두산디지털이노베이션 등

### 📍 박 성훈 — GIS·공간데이터 분석

- **분석 목표**: GIS·공간데이터 분석가의 채용 시장 규모와 취준생 준비 동향 분석
- **핵심 키워드**: GIS, 공간데이터, 측량, 지리정보, QGIS
- **대상 기업(예시)**: 한국국토정보공사, 새한항업, iGiS, 공간정보 기업 등

### 🤖 박 준영 — 자율주행·의료 AI

- **분석 목표**: 자율주행·의료 AI 분야에서 요구하는 기술스택과 자격 조건은 무엇이며, 취준생은 어떻게 준비하고 있는가?
- **핵심 키워드**: 자율주행, ADAS, 의료 AI, 헬스케어, PyTorch, ROS
- **대상 기업(예시)**: 뷰노, 루닛, 딥노이드, 현대모비스, 네이버랩스 등

### 🚗 정 유진 — 자동차·모빌리티 SW

- **분석 목표**: 자동차·모빌리티 산업 SW 개발자의 근무 환경과 기업 실태 비교 분석
- **핵심 키워드**: 자동차 SW, 임베디드, 커넥티드카, 모빌리티
- **대상 기업(예시)**: 현대차, 기아, 현대모비스, 현대오토에버, 모빌리티 스타트업 등

## 5. 공통 분석 프레임워크

모든 팀원이 **동일한 기준**으로 데이터를 수집하여, 최종 발표에서 4개 분야 간 비교가 가능하도록 합니다.

| #   | 분석 항목            | 설명                              | 주요 출처         |
| --- | -------------------- | --------------------------------- | ----------------- |
| ①   | 채용 공고 수         | 키워드별 공고 건수 추이           | 사람인 / 원티드   |
| ②   | 요구 기술스택 TOP 10 | 프로그래밍 언어, 프레임워크, 도구 | 채용 공고 텍스트  |
| ③   | 자격증·우대사항      | 어떤 자격증을 요구/우대하는지     | 채용 공고 텍스트  |
| ④   | 경력/신입 비율       | 신입 채용 가능성                  | 채용 공고 필터    |
| ⑤   | 연봉 수준            | 분야별 평균 연봉 비교             | 잡플래닛 / 사람인 |
| ⑥   | 기업 평점·복지       | 근무 만족도 정량화                | 잡플래닛          |
| ⑦   | 취준 트렌드          | 블로그에서 언급되는 준비 내용     | Naver OpenAPI     |
