## 1. 프로젝트 개요

### 1.1 분석 목적

본 프로젝트는 국내 자율주행/모빌리티 산업의 **취업 동향**을 데이터 기반으로 파악하는 것을 목적으로 한다. 구직자 관점의 블로그 콘텐츠, 산업 동향 뉴스, 실제 채용 공고, 재직자 기업 리뷰를 종합하여 다음 질문에 답한다.

- 자율주행 분야에서 가장 많이 요구하는 **기술스택**은 무엇인가?
- 채용 시장의 **직무 분포**와 **근무지 편중**은 어떠한가?
- 자율주행 5대 도메인(인지/판단/제어/인프라/시뮬레이션) 중 채용 수요가 높은 영역은?
- 기업별 **면접 난이도**와 **직원 만족도**는 어떠한가?
- 블로그/뉴스 콘텐츠의 **시계열 트렌드**는 어떤 패턴을 보이는가?

### 2.2 원티드 API 크롤링

| 항목 | 내용 |
|------|------|
| **수집 방법** | Wanted v4 REST API (비공식) |
| **카테고리** | 모빌리티 (CATEGORY_ID = 660) |

| **수집 항목** | 포지션명, 회사명, 업종, 근무지, 연봉범위, 주요업무, 자격요건, 우대사항, 기술스택태그 등 16개 필드 |

#### 왜 이 방법을 사용했는가

- **구조화된 채용 데이터**: 원티드는 채용 공고별 기술스택 태그, 자격요건, 우대사항이 체계적으로 구분되어 있어 정량 분석에 적합하다.
- **API 방식 채택**: Selenium 대신 HTTP API를 직접 호출하여 수집 속도가 빠르고 안정적이다. 원티드 v4 API는 JSON 형식으로 목록 조회(offset 기반 페이징) → 개별 상세 조회 2단계로 구성된다.
- **카테고리 필터링**: CATEGORY_ID 660(모빌리티)을 지정하여 자율주행/모빌리티 직군 포지션만 타겟 수집하였다.
- **직무 서브카테고리 매핑**: 자율주행 엔지니어, ADAS 엔지니어, 차량제어 엔지니어, 센서융합 엔지니어, HD맵 엔지니어, V2X 통신 엔지니어 등 21개 세부 직무를 정의하여 분류하였다.

### 2.3 잡플래닛 크롤링

| 항목 | 내용 |
|------|------|
| **수집 방법** | Selenium WebDriver (Chrome headless) |
| **인증** | 로그인 필수 (ID/PW) |
| **대상 기업** | 14개 자율주행/모빌리티 기업 |
| **수집 건수** | 기업정보 14건, 리뷰 42건, 면접후기 19건 |
| **수집 항목** | 기업별 별점, 리뷰(장점/단점/별점/직종/재직상태), 면접후기(난이도/경로/질문) |

#### 왜 이 방법을 사용했는가

- **Selenium 필수**: 잡플래닛은 로그인 후에만 리뷰/면접후기를 볼 수 있고, JavaScript로 동적 렌더링되는 SPA 구조이므로 단순 HTTP 요청으로는 데이터 수집이 불가능하다. Selenium을 통해 실제 브라우저를 자동화하여 로그인 → 페이지 탐색 → 데이터 추출 과정을 수행하였다.
- **정성적 데이터의 가치**: 잡플래닛의 기업 리뷰와 면접후기는 실제 재직자/면접 경험자가 작성한 **1차 자료**로, 채용 공고만으로는 파악할 수 없는 기업 내부 문화, 면접 난이도, 장단점을 보여준다.
- **안티봇 대응**: Chrome WebDriver 탐지를 우회하기 위해 `--disable-blink-features=AutomationControlled` 옵션, User-Agent 설정, 랜덤 딜레이(2~5초)를 적용하였다.

#### 결과 해석

- **기술스택 워드클라우드 (`wc_tech_stack.png`)**: Java, AWS, Python이 가장 크게 표시되어 범용 기술의 우위를 보여준다. 동시에 ADAS, V2X, AUTOSAR, LiDAR, ROS 등 자율주행 특화 기술도 뚜렷하게 나타나, 이 분야가 **범용 SW + 도메인 전문성**을 함께 요구함을 시각적으로 확인할 수 있다.
- **블로그 키워드 워드클라우드 (`wc_blog_keywords.png`)**: "자율주행"이 압도적으로 크며, "개발자", "취업", "자동차", "AI", "로봇" 등이 뒤따른다. 구직자들이 자율주행 외에도 로봇/AI 분야와 병행하여 취업을 준비하는 경향을 보여준다.
- **도메인 키워드 워드클라우드 (`wc_domain.png`)**: "자율주행", "개발자", "AI", "과정" 등이 두드러지며, 교육과정이나 부트캠프를 통한 역량 개발에 대한 관심이 높음을 나타낸다.
- **채용 자격요건 워드클라우드 (`wc_job_requirements.png`)**: 원티드 채용공고의 자격요건/우대사항에서 추출한 키워드로, 실제 채용 시장의 요구사항을 반영한다.

### 5.3 고급 차트 (10종)

#### 5.3.1 키워드-기술 히트맵 (`adv_keyword_tech_heatmap.png`)

- **결과 해석**: 특정 키워드(예: "AUTOSAR 개발자")에서 해당 기술의 언급 빈도가 집중되어, 키워드-기술 간 강한 상관관계를 보여준다.

#### 5.3.2 기업 종합 레이더차트 (`adv_company_radar.png`)

- **사용 이유**: 레이더차트는 **다차원 지표를 하나의 도형으로 비교**하여 기업별 프로필을 직관적으로 비교할 수 있다. 리뷰수, 평균별점, 직종다양성, 면접난이도, 면접후기수 5개 축을 사용하였다.

#### 5.3.3 도메인별 키워드 빈도 히트맵 (`adv_domain_keyword_heatmap.png`)

- **사용 이유**: 자율주행 5대 도메인(인지/판단/제어/인프라/시뮬레이션) × 개별 기술 키워드의 교차 빈도를 시각화하여 **도메인-기술 매핑**을 명확히 한다.

#### 5.3.4 기술스택 동시출현 네트워크 (`adv_tech_network.png`)
