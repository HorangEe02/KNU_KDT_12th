## 1. 프로젝트 개요

### 1.1 분석 목적

의료 AI 분야는 디지털 헬스케어 시대의 핵심 산업으로 급성장하고 있다. 본 프로젝트는 **의료 AI 분야 취업을 준비하는 구직자**의 관점에서 채용 시장의 현황과 트렌드를 데이터 기반으로 파악하기 위해 수행되었다. 구체적으로 다음을 목표로 한다:

- 의료 AI 분야에서 **어떤 기술 스택이 요구**되는지 파악
- 주요 **의료 AI 기업들의 채용 현황과 기업 문화** 분석
- 취업 준비에 필요한 **자격증, 학력, 경력 요건** 도출
- 블로그·뉴스를 통한 **취업 트렌드 및 시장 동향** 파악

### 1.2 분석 파이프라인

전체 파이프라인은 **5단계**로 구성된다:

```
[Phase 1] 크롤링 → [Phase 2] 데이터 전처리 → [Phase 3] 텍스트 분석 → [Phase 4] 시각화 → [Phase 5] URL 정리
```

| 단계 | 내용 | 산출물 |
|------|------|--------|
| Phase 1 | 4개 소스에서 데이터 수집 | CSV/JSON 원본 데이터 (9개 파일) |
| Phase 2 | HTML 제거, 정규화, 중복 제거 | 전처리 완료 CSV (11개 파일) |
| Phase 3 | 형태소 분석, 빈도 분석 | 기술스택/자격증/도메인 빈도 CSV |
| Phase 4 | 10개 시각화 모듈 실행 | 64개 차트 (PNG 62 + HTML 2) |
| Phase 5 | 수집 URL 정리 | crawling_urls.csv (3,262건) |

### 1.3 데이터 소스 개요

| 데이터 소스 | 수집 방식 | 수집 유형 | 선택 이유 |
|------------|----------|----------|----------|
| **네이버 OpenAPI** | REST API (urllib) | 블로그 532건, 뉴스 451건 | 국내 최대 포털로 취업 준비 관련 블로그·뉴스가 가장 풍부 |
| **사람인** | 웹 크롤링 (requests + BS4) | 채용공고 2,078건 | 국내 최대 취업 포털로 다양한 규모의 기업 공고 포함 |
| **원티드 API** | REST API v4 (requests) | 채용공고 75건 | IT/스타트업 중심 채용 플랫폼으로 의료 AI 스타트업 공고 집중 |

### 2.3 사람인 채용공고 크롤링 (2,078건)

#### 크롤링 이유
사람인은 **국내 최대 규모의 종합 취업 포털**로, 의료 AI 분야의 대기업부터 스타트업까지 폭넓은 채용 공고를 게시한다. 실제 채용 시장에서 요구하는 기술 스택, 경력 조건, 학력 요건 등을 정량적으로 파악하기 위해 수집하였다.

#### 수집 방식
- **requests + BeautifulSoup** 기반 검색 결과 카드(`.item_recruit`) 파싱
- 사람인 상세 페이지는 JavaScript 렌더링으로 접근이 제한되어, 검색 결과 카드에서 직접 데이터 추출
- 키워드당 최대 5페이지씩 순회하며 채용 카드 파싱
- 기술스택/자격증은 공고제목 + 직무분야 텍스트에서 정규식으로 추출

#### 검색 키워드 설계 (12개)

| 키워드 | 설계 의도 |
|--------|----------|
| "의료 AI", "의료 인공지능" | 핵심 분야 직접 검색 |
| "헬스케어 AI", "디지털헬스" | 유사 분야 확장 검색 |
| "의료영상 분석", "의료 딥러닝", "의료 머신러닝" | 기술 특화 키워드 |
| "헬스케어 Python", "의료 데이터" | 개발 기술 + 의료 조합 |
| "바이오 AI" | 바이오/제약 융합 검색 |
| "PACS 개발", "EMR 개발" | 의료 IT 인프라 특화 |

#### 수집 결과
- **2,078건**의 채용 공고 수집 (12개 키워드, 중복 제거 완료)
- 수집 필드: 공고제목, 회사명, 경력조건, 학력조건, 근무지역, 고용형태, 직무분야, 마감일, 등록일, 기술스택, 자격증, 배지, 검색키워드
- **전체 데이터의 63%**를 차지하는 최대 규모 데이터 소스

### 2.4 원티드 채용공고 크롤링 (75건)

#### 크롤링 이유
원티드는 **IT/스타트업 중심의 채용 플랫폼**으로, 의료 AI 스타트업들이 집중적으로 채용 공고를 게시한다. 특히 원티드는 **기술 스택(skill_tags)**을 구조화하여 제공하므로, 채용 시장에서 요구하는 기술을 정량적으로 분석하기에 최적이다.

### 2.5 잡플래닛 크롤링 (기업정보 12건 + 리뷰 155건 + 면접후기 101건)

- **기업 내부 문화**: 장점/단점, 경영진 평가, 워라밸 등
- **면접 과정**: 면접 질문, 난이도, 합격/불합격 경험
- **연봉 정보**: 직급별 연봉 수준

#### 수집 방식

- 회원제 열람 제한(Paywall)을 고려하여 로그인 기반 크롤링 수행
- 기업 간 **3~6초 랜덤 딜레이**로 서버 부하 최소화

## 3. 대상 기업 선정 이유 및 과정

### 3.1 선정 기준

12개 대상 기업은 다음 **4가지 기준**을 종합하여 선정하였다:

1. **의료 AI 핵심 기업**: 의료 AI를 핵심 사업으로 영위하는 기업
2. **카테고리 다양성**: 의료영상, 디지털헬스, 원격의료, 플랫폼 등 다양한 세부 분야 포함
3. **기업 규모 다양성**: 스타트업부터 대기업까지 포함하여 취업 시장 전체를 조망
4. **시장 영향력**: 업계에서의 인지도, 투자 유치, FDA/MFDS 인허가 실적 등

### 3.2 기업별 선정 이유

#### 스타트업 - 의료영상 AI (3개사)

| 기업명 | 선정 이유 |
|--------|----------|
| **뷰노** | 국내 의료영상 AI 선두주자. 골연령 판독 AI `BoneAge`, 흉부 X-ray AI 등 **다수의 MFDS 인허가** 획득. 코스닥 상장사로 의료 AI 스타트업의 대표 기업 |
| **루닛** | 흉부 X-ray AI `Lunit INSIGHT CXR`로 **FDA 승인** 획득. 글로벌 시장에서 가장 활발하게 진출 중인 국내 의료영상 AI 기업 |
| **딥노이드** | 의료영상 딥러닝 전문기업. 뇌 MRI, 척추, 폐 등 **다양한 영상 모달리티**를 커버하여 기술 다양성 대표 |

**선정 과정**: 의료영상 AI는 의료 AI 분야에서 가장 상용화가 활발한 영역이다. MFDS/FDA 인허가를 획득한 제품을 보유한 기업 중 시장 점유율과 기술력이 검증된 3개사를 선정하였다.

#### 스타트업 - 의료AI/디지털헬스 (3개사)

| 기업명 | 선정 이유 |
|--------|----------|
| **제이엘케이** | 뇌졸중 AI 진단 솔루션. 의료영상 외 **임상 의사결정 지원** 영역을 대표하는 기업 |
| **라인웍스** | **의료 자연어처리(NLP)** 전문. 비정형 의료 데이터(진료기록 등) 분석이라는 독특한 포지션 |
| **메디컬에이아이** | **심전도 AI 분석** 전문. 영상의학이 아닌 **생체신호 분석** 분야를 대표 |

**선정 과정**: 의료 AI는 영상 분석에만 국한되지 않는다. 임상 의사결정, 자연어처리, 생체신호 분석 등 다양한 기술 영역을 포함하기 위해 각 분야의 대표 기업을 1개씩 선정하였다.

#### 스타트업 - 원격의료 (1개사)

| 기업명 | 선정 이유 |

### 6.2 핵심 인사이트

#### 1. Python + AI/ML이 실제 채용 시장의 핵심 기술

사람인 공고제목/직무분야 텍스트까지 반영한 결과, Python(790건)이 압도적 1위, 딥러닝(598건)·머신러닝(585건)이 2~3위를 차지하였다. **실제 채용 시장에서는 Python 기반 AI/ML 개발 역량이 가장 높은 수요**를 보인다.

#### 2. 도메인 지식은 여전히 핵심 차별화 요인

R(535건), 의료영상(245건), 임상시험(195건), AWS(141건) 등 도메인 특화 기술/지식과 클라우드 인프라가 상위 15위 안에 다수 포진하였다. 순수 개발 역량만으로는 부족하며, **의료영상, 임상시험, 인허가 등 도메인 전문성**이 경쟁력의 핵심이다.

#### 3. 의료 면허 + IT 기술의 융합 인재 수요 급증

자격증 분석에서 방사선사(34건, 46.58%)가 압도적 1위, 임상병리사(14건)가 2위를 차지하였다. 이는 **의료 현장의 도메인 지식을 갖춘 인력**이 AI 기술을 접목할 때 높은 가치를 인정받는다는 것을 의미한다.

#### 4. 영상의학이 채용 시장의 핵심 도메인

도메인 분석에서 영상의학(572건, 53%)이 압도적 1위를 차지하였다. 의료영상(245건), CT(239건) 등이 채용공고에서 대거 탐지되어, **의료영상 AI가 채용 시장에서 가장 큰 비중**을 차지함을 확인하였다.

#### 5. 수도권 집중 현상 확인

사람인, 원티드, 잡플래닛 데이터 모두에서 서울(특히 강남/판교)에 의료 AI 기업이 집중되어 있는 패턴을 확인하였다.
