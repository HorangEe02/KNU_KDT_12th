# 잡플래닛 크롤링 (기업정보 12건 + 리뷰 155건 + 면접후기 101건)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **데이터 소스** | 잡플래닛 (jobplanet.co.kr) |
| **수집 건수** | 기업정보 12건 + 리뷰 155건 + 면접후기 101건 (총 268건) |
| **수집 목적** | 의료 AI 기업 내부 정보 파악 (평점, 직원 리뷰, 면접 난이도/경로 등) |
| **크롤링 방식** | Selenium (Chrome WebDriver) + BeautifulSoup |

## 2. 수집 방식

### 2.1 사용 기술
- **브라우저 자동화**: Selenium (Chrome WebDriver)
- **HTML 파서**: BeautifulSoup
- **로그인 필요**: 잡플래닛 계정 로그인 후 크롤링 (JavaScript 기반 로그인)
- **구현 파일**: `crawlers/jobplanet_crawler.py` > `JobPlanetCrawler` 클래스

### 2.2 수집 흐름

```
잡플래닛 로그인 (Selenium)
    ↓
대상 기업 12개 순회
    ├─ 기업 기본 정보 수집 (/companies/{id})
    ├─ 직원 리뷰 수집 (/companies/{id}/reviews, 최대 3페이지)
    ├─ 연봉 정보 수집 (/companies/{id}/salaries)
    └─ 면접 후기 수집 (/companies/{id}/interviews, 최대 2페이지)
    ↓
기업별 3~6초 랜덤 딜레이
    ↓
CSV 저장 (data/raw/)
```

### 2.3 안티봇 대응
- `--disable-blink-features=AutomationControlled` 옵션
- `navigator.webdriver` 속성 undefined 처리
- User-Agent 설정
- 기업 간 3~6초 랜덤 딜레이

## 4. 크롤링 상세 로직

### 4.1 로그인

```
잡플래닛 로그인 페이지 접속
    ↓
이메일 입력 (JavaScript 기반, 다중 셀렉터 시도)
    ↓
비밀번호 입력 (JavaScript 기반)
    ↓
로그인 버튼 클릭
    ↓
URL 확인으로 로그인 성공 여부 판단
```

- `.env` 파일의 `JOBPLANET_USER_ID` / `JOBPLANET_PASSWORD` 사용
- 로그인 실패 시에도 제한적으로 크롤링 진행

### 4.2 기업 검색 (company_id 자동 탐색)

- `settings.py`에 `company_id`가 지정되어 있으면 직접 사용
- 미지정 시 기업명으로 검색 → `ul.grid` 내 검색 결과에서 매칭
- "AI" 포함 기업명은 "에이아이", 공백 제거 등 변형 검색 시도

### 4.3 리뷰 크롤링

- React/Tailwind 신규 레이아웃 대응
- 리뷰 카드에서 "전직원"/"현직원" 키워드로 카드 식별
- `span.text-body2` 요소에서 직종, 재직상태, 위치, 등록일 추출
- 리뷰 본문은 멤버십 벽 뒤에 있어 제한적 수집

### 4.4 면접후기 크롤링

- `section.content_ty4` 레거시 레이아웃 대응
- `dt/dd` 쌍에서 면접난이도, 면접일자, 면접경로 추출
- 면접경험 본문은 블라인드 커버 없는 경우만 수집

## 6. 데이터 특성

- **로그인 필수**: 잡플래닛 계정 로그인이 필요하며, 로그인 없이는 데이터 접근이 제한됨
- **멤버십 벽**: 리뷰 본문, 면접 경험 상세 내용은 유료 멤버십 구독 없이 접근 불가
- **수집 제한**: 리뷰 최대 3페이지, 면접후기 최대 2페이지로 제한
- **동적 렌더링**: React/Tailwind 기반 신규 레이아웃과 레거시 레이아웃이 혼재
- **CAPTCHA 위험**: 과도한 요청 시 CAPTCHA 인증이 요구될 수 있음
- **데이터 한계**:
  - 리뷰 본문(장점, 단점, 경영진에 바라는 점)은 멤버십 벽으로 대부분 미수집
  - 연봉 상세 정보는 멤버십 벽 제한으로 제한적 수집
  - 12개 타겟 기업에 한정된 수집 (전체 의료 AI 기업 대상이 아님)
