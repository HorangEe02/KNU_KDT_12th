## 1. 기(起) - 프로젝트의 시작

### 1.1 왜 이 프로젝트를 수행하게 되었는가

본 프로젝트는 KDT 12기 교육과정에서 **현재까지 학습한 데이터 전처리 및 시각화 역량을 실제 데이터에 적용**하기 위해 기획되었다. 단순히 연습용 데이터가 아닌, 실전에서 의미 있는 분석을 수행함으로써 학습 내용의 체화를 목표로 하였다.

주제 선정에 있어 다음 세 가지 기준을 고려하였다.

1. **실전 적용에 가장 적합한 분야**로 의료 데이터를 선정하였다. 의료 데이터는 변수 간 관계가 복잡하고, 교란변수 통제가 필수적이며, 분석 결과의 해석이 사회적으로 의미가 있기 때문이다.
2. **'백해무익'이라 불리는 흡연**이 실제로 우리 몸에 미치는 영향을 정량적으로 검증하고자 하였다. 누구나 "흡연은 건강에 해롭다"고 알고 있지만, 이를 데이터 기반으로 수치화하여 입증하는 것은 다른 차원의 작업이다.
3. **대규모 공개 데이터셋의 활용 경험**을 쌓고자 하였다. CDC BRFSS 데이터(319,795건)는 실제 미국 질병통제예방센터에서 수집한 인구 기반 설문조사 데이터로, 실무에서 다루게 될 대용량 데이터 처리를 미리 경험할 수 있었다.

### 1.2 연구 질문과 가설

핵심 연구 질문은 다음과 같다.

> *"흡연은 나이, 성별, BMI, 음주, 신체활동 등 다른 요인들을 고려한 후에도 뇌졸중 발생 위험을 높이는가?"*

이 질문에 답하기 위해 세 가지 가설을 설정하였다.

| 구분 | 귀무가설 (H0) | 대립가설 (H1) |
|------|-------------|-------------|
| **가설 1** | 흡연자와 비흡연자 간 뇌졸중 발생률에 차이가 없다 | 흡연자는 비흡연자보다 뇌졸중 발생률이 높다 |
| **가설 2** | 교란변수 보정 후 흡연과 뇌졸중 간 연관성이 사라진다 | 교란변수 보정 후에도 흡연과 뇌졸중 간 연관성이 유의하게 유지된다 |
| **가설 3** | 흡연 여부와 신체활동 수준 간 상호작용 효과가 없다 | 흡연과 신체활동 부족이 결합될 때 뇌졸중 위험이 각 요인의 개별 효과를 합한 것보다 크다 |

## 2. 승(承) - 프로젝트의 전개

### 2.1 데이터 소개

미국 질병통제예방센터(CDC)의 행동위험요인감시체계(BRFSS) 2020년 데이터를 사용하였다. Kaggle에서 공개된 "Indicators of Heart Disease" 데이터셋으로, 원본 BRFSS에서 심혈관 질환 관련 주요 변수를 추출하여 정제한 것이다.

| 항목 | 내용 |
|------|------|
| 총 샘플 수 | 319,795명 |
| 변수 수 | 18개 (수치형 4개, 범주형 14개) |
| 결측치 | 0건 (100% 완성도) |
| 메모리 사용량 | 약 232.85 MB |
| 종속변수 (Y) | Stroke - 뇌졸중 진단 여부 (Yes/No) |
| 독립변수 (X) | Smoking - 평생 100개비 이상 흡연 여부 (Yes/No) |
| 교란변수 | Age, Sex, BMI, PhysicalActivity, AlcoholDrinking |

### 2.2 프로젝트 워크플로우

전체 분석은 4단계(STEP)로 구성되었다.

```
STEP 1. 데이터 수집        STEP 2. 데이터 탐색 및 전처리
 - 분석 목적에 맞는          - 변수 유형 구분
   데이터셋 선정               (수치형 4개, 범주형 14개)
 - 분석 환경에 데이터         - 결측치 탐지 및 처리
   로드 및 기본 구조 확인     - 이상치 탐지 및 처리 여부 결정
        |                          |
        v                          v
STEP 3. 데이터 시각화        STEP 4. 결론 도출
 - 주요 변수 분포 시각화      - 시각화를 통해서 얻은
 - 클래스 불균형 확인           분석 결과 분석
 - 흡연 상태별 뇌졸중         - 자체 평가 의견
   발생률 비교
```

### 2.3 데이터 전처리

#### 범주형 변수 처리
- **이진 변수**: Yes=1, No=0으로 인코딩 (Stroke, Smoking, PhysicalActivity 등)
- **성별**: Male=1, Female=0 (더미 변수화하여 회귀분석에서 남성일 때의 효과를 직접 측정)
- **나이 범주**: 각 범주의 중앙값을 사용하여 수치형으로 변환 (예: '18-24' → 21, '80 or older' → 82)
- **건강 상태**: 5단계 서열 척도로 수치화 (Poor=1, Fair=2, Good=3, Very good=4, Excellent=5)

#### 이상치 처리
BMI 변수에서 IQR 방법 적용 시 3.3%(10,396건)의 이상치가 탐지되었으나, 의학적으로 가능한 범위(12.0~94.8) 내에 있어 **분석에 포함하기로 결정**하였다.

### 2.4 분석 방법론

본 프로젝트에서는 전통적 통계 분석과 머신러닝 기법을 병행하였다.

#### 통계 분석
| 방법 | 목적 |
|------|------|
| 카이제곱 검정 (Chi-square test) | 흡연과 뇌졸중 간 독립성 검정 |
| Cramer's V | 효과 크기 측정 |
| Mann-Whitney U 검정 | BMI와 뇌졸중의 관계 분석 |
| 다변량 로지스틱 회귀분석 | 교란변수 단계적 보정 |
| 분산팽창지수 (VIF) | 다중공선성 진단 |

#### 머신러닝 모델
| 모델 | 유형 |
|------|------|
| Logistic Regression | 기준 모델 |
| Random Forest | 앙상블 (Bagging) |
| XGBoost | 앙상블 (Gradient Boosting) |

- **클래스 불균형 처리**: SMOTE(Synthetic Minority Over-sampling Technique) 적용
- **검증 방법**: Stratified 5-Fold Cross-Validation
- **평가 지표**: Accuracy, Precision, Recall, F1-Score, ROC-AUC

## 3. 전(轉) - 분석 결과와 발견

### 3.1 기술통계: 흡연자는 정말 뇌졸중에 더 잘 걸리는가?

전체 319,795명 중 뇌졸중 경험자는 12,090명(3.8%)으로 **클래스 불균형(1:25)**이 심각하였다. 흡연 경험자는 131,908명(41.2%)이었다.

흡연 여부에 따른 뇌졸중 발생률을 비교한 결과:

| 그룹 | 뇌졸중 No | 뇌졸중 Yes | 합계 | **발생률** |
|------|---------|----------|------|----------|
| 비흡연 | 182,633 | 5,254 | 187,887 | **2.80%** |
| 흡연 | 125,093 | 6,815 | 131,908 | **5.17%** |

**흡연자의 뇌졸중 발생률은 비흡연자의 약 1.85배**로 나타났다.

### 3.2 카이제곱 검정: 통계적으로 유의한가?

| 통계량 | 값 | 해석 |
|--------|-----|------|
| Chi-square (χ²) | 536.89 | 매우 큰 값 |
| 자유도 (df) | 1 | - |
| p-value | < 0.001 | 유의수준 0.05보다 훨씬 작음 |
| Cramer's V | 0.041 | 효과 크기는 작은 수준 |

p-value < 0.001로 **귀무가설을 기각**하여, 흡연과 뇌졸중 사이에 통계적으로 유의한 연관성이 존재함을 확인하였다. 다만 Cramer's V가 0.041로 효과 크기 자체는 작았다.

> **가설 1 검증 결과: 지지됨** - 흡연자는 비흡연자보다 뇌졸중 발생률이 유의하게 높다.

### 3.3 다변량 로지스틱 회귀: 교란변수를 보정하면 어떻게 되는가?

교란변수를 단계적으로 보정하며 흡연의 오즈비(OR) 변화를 추적하였다.

| 모델 | 보정 변수 | OR | 95% CI | p-value |
|------|---------|-----|--------|---------|
| Model 1 (Crude) | 없음 | **1.53** | [1.48, 1.58] | < 0.001 |
| Model 2 | 나이, 성별 | **1.08** | [1.04, 1.12] | < 0.001 |
| Model 3 (Full) | 나이, 성별, BMI, 신체활동, 음주 | **1.06** | [1.02, 1.10] | 0.002 |

**핵심 발견**: 비보정 OR(1.53)에서 나이/성별 보정 후 OR(1.08)로 급격히 감소하였다. 이는 **나이가 흡연과 뇌졸중 양쪽 모두와 강하게 연관된 주요 교란변수**임을 의미한다. 그러나 모든 교란변수를 보정한 후에도 OR=1.06으로 **여전히 통계적으로 유의**하였다.

Full Model에서 각 변수의 영향력:

| 변수 | OR | 해석 |
|------|-----|------|
| **Age (10세 증가당)** | 2.15 | 가장 강력한 위험 요인 |
| Sex (Male) | 1.12 | 남성이 약간 높은 위험 |
| **Smoking** | 1.06 | 독립적 위험 요인 (보정 후에도 유의) |
| BMI (5단위 증가당) | 1.03 | 약한 위험 요인 |
| **PhysicalActivity** | 0.65 | 보호 요인 (35% 위험 감소) |
| AlcoholDrinking | 0.68 | 보호 요인 |

> **가설 2 검증 결과: 지지됨** - 교란변수 보정 후에도 흡연과 뇌졸중의 연관성이 유의하게 유지된다.

### 3.4 상호작용 분석: 흡연 + 신체활동 부족은 얼마나 위험한가?

흡연 여부와 신체활동 여부를 조합한 4분류 분석 결과:

| 그룹 | 뇌졸중 발생률 | 상대 위험도 (RR) |
|------|-------------|---------------|
| 비흡연 + 활동적 (참조군) | **2.23%** | 1.00x |
| 비흡연 + 비활동적 | 5.19% | 2.33x |
| 흡연 + 활동적 | 4.11% | 1.84x |
| **흡연 + 비활동적** | **7.97%** | **3.57x** |

**최고 위험군인 '흡연 + 비활동적' 그룹**은 참조군 대비 **3.57배** 높은 뇌졸중 발생률을 보였다. 주목할 점은, 흡연자라 하더라도 운동을 하면 발생률이 7.97%에서 4.11%로 **약 48% 감소**한다는 것이다. 이는 **신체활동이 흡연의 부정적 영향을 부분적으로 상쇄**할 수 있음을 시사한다.

연령대별로 보면 18-24세에서 ~0.3%(비흡연)/~0.6%(흡연)이던 발생률이 80세 이상에서 ~9.0%/~10.3%까지 급상승하여, 나이의 강력한 영향을 재확인할 수 있었다.

> **가설 3 검증 결과: 부분적으로 지지됨** - 흡연 + 비활동 그룹이 가장 높은 위험을 보였으나, 상호작용 항 자체는 통계적으로 유의하지 않았다(p=0.087).

| Random Forest | 0.765 | 0.132 | 0.642 | 0.219 | 0.802 |

| 6 | AlcoholDrinking | 0.046 |

나이(Age)가 전체 예측력의 45.2%를 차지하여 **압도적으로 가장 중요한 변수**임을 재확인하였다.

### 3.7 종합 결론

| 핵심 발견 | 내용 |
|---------|------|
| 흡연의 위험성 | 흡연은 성별보다 강력한 뇌졸중 위험 요인이며, 비흡연자 대비 발생률이 유의하게 높다 |
| 연령 취약성 | 나이가 가장 강력한 위험 요인(r=0.14)으로, 고령층 대상 집중 관리가 필요하다 |
| 운동의 보호 효과 | 신체활동은 뇌졸중 위험을 낮추는 보호 요인이며, 흡연자도 운동 시 발생률이 48% 감소한다 |
| 복합 관리 | 금연과 규칙적 운동의 병행이 뇌졸중 예방에 가장 효과적인 전략이다 |

### 4.2 결과 해석에 대한 평가

**통계 분석 결과의 신뢰성**
- 319,795건의 대규모 표본으로 통계적 검정력은 충분하였다.
- 그러나 효과 크기(Cramer's V=0.041, Adjusted OR=1.06)가 작아, **통계적 유의성과 임상적 유의성 사이의 차이**를 인식하는 것이 중요하다. 대규모 데이터에서는 작은 차이도 통계적으로 유의하게 나올 수 있기 때문이다.
- 비보정 OR(1.53)에서 보정 후 OR(1.06)으로의 급격한 감소는 **나이라는 교란변수의 압도적 영향**을 보여주며, 교란변수 통제의 중요성을 실감할 수 있었다.

**머신러닝 결과의 한계**
- ROC-AUC 0.79~0.81 수준은 "acceptable" 수준이나 "good"(0.8~0.9)에는 미치지 못했다.
- 클래스 불균형(뇌졸중 3.8%)으로 인해 Precision이 0.12~0.14로 매우 낮아, **실무 적용에는 한계**가 있다.
- SMOTE를 적용하였으나 더 정교한 방법(Focal Loss, Cost-sensitive Learning)의 효과를 충분히 검증하지 못하였다.

### 4.3 느낀 점 - 자아성찰

**박준영**
> 단순히 코드를 수정하는 것이 아닌, **데이터의 구조적 특성을 다시 들여다보는 과정**이었다. 같은 결과를 보여줄 때도 시각화 방법이 다양하다는 사실을 알게 되었다.

**이창민**
> **판다스를 활용하는 능력과 창의력을 조금 더 키울 필요**가 느끼는 과정이었다. 시간만 더 존재했으면 다른 연관성에 대해서 생각할 수 있었을 것 같다.
